Notes

When MI takes a snapshot of this page, the browser will crop around the "#mi_size_container" element. This can be any ID you want it to be, and is modifiable in the Movable Ink application when setting up the crop in the "CSS Selector" field. Everything outside of this DOM element (that is not overlapping) will not be included in the crop.

View page source and take a look at the comments for the CSS for some more info about CSS 3 support.
